Featured Projects

Green AI: Benchmarking Local LLM/SLM Energy Consumption, CodeCarbon vs pyJoules

This is a series of scripts that estimate the energy consumption of local inference on popular open-source SLMs that are available on Ollama. The ultimate goal is to compare the estimation from CodeCarbon with that from pyJoles. Both Python libraries use Intel RAPL and NVML to read data directly from the CPU and GPU. A Bash script was used to automate the process of creating virtual environments, verifying dependencies, and launching the main Python scripts. Another Python script was used to create and elaborate plots.
